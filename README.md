# Climate-Resilient Crop Recommendation System

A machine learning-based system for recommending crops using soil and environmental data.

## Overview

Models used:

* Decision Tree
* Random Forest
* SVM
* Gradient Boosting

## Dataset

7,630 samples with 14 agro-climatic features across 12 crop classes 

## Results

* Random Forest achieves highest accuracy (84.33%)
* Soil nutrients are the most influential features (via SHAP)
* Random Forest shows strong robustness to climate variations
